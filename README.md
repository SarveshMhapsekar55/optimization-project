# Optimization Model - Profit Maximization

## Objective

To solve a business problem using optimization techniques and Python (PuLP).

## Problem Statement

A company produces two products with limited resources such as labor and materials. The goal is to maximize profit.

## Method Used

Linear Programming using PuLP library.

## Model Details

* Objective Function: Maximize 20x + 30y
* Constraints:

  * 2x + y ≤ 40
  * x + 2y ≤ 50

## Results

* Product A: 10 units
* Product B: 20 units
* Maximum Profit: 800

## Tools Used

* Python
* PuLP

## Conclusion
Optimization techniques help businesses make efficient decisions under constraints.
